# Live Poll App

A real-time polling application that allows users to create, share, and participate in live polls.

## Features

- Create custom polls with multiple options
- Real-time vote counting and updates
- Share polls via shareable links
- View live results as votes come in
- Clean and responsive UI

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js/Express
- **Database**: MongoDB

## Installation

```bash
git clone https://github.com/Ajmalka84/live-poll-app.git
cd live-poll-app
npm install
```

## Usage

```bash
npm start
```

Visit `http://localhost:3000` in your browser.
